Convolutional Neural Network Based Technique for Flower Classification

Author: Yang Haoyun (Ethan)

Abstract

This project explores flower image classification using convolutional neural networks (CNNs), particularly the VGG16 model. By training on the Oxford 17 flower dataset, the model achieved a classification accuracy of 92.04%. The model architecture includes 13 convolutional layers, 5 pooling layers, and 3 fully connected layers followed by a softmax output. ReLU was used as the activation function, and SGD was employed for optimization.

Introduction

With increasing image data availability, efficient and accurate flower species classification is essential. Traditional manual feature extraction methods have limitations in accuracy and efficiency. Convolutional neural networks offer an automated approach to feature extraction, improving classification performance. VGGNet, developed by Oxford University, enhances classification through deeper layers and smaller convolutional kernels.

Aim and Objectives

Aim: Develop a fast and accurate flower classification system using convolutional neural networks.

Objectives:

  • Investigate CNN models for flower classification
  • Select and preprocess suitable datasets
  • Design and train a VGG16-based CNN model
  • Develop a GUI for dataset selection, training, and prediction
  • Evaluate the model using accuracy and loss metrics

Background Review

Related studies have used various CNN architectures such as AlexNet, ResNet, and InceptionV3, alongside data preprocessing techniques to improve recognition accuracy. Techniques like superpixel segmentation, transfer learning, and saliency mapping have proven effective in enhancing model performance.

Methodology

The VGG16 model was employed with image preprocessing to match the required 224x224 input size. The training dataset consists of 1088 images, with 272 images for testing. Python, PyTorch, and PyQt5 were used to implement model training, evaluation, and GUI development.

Technologies Used

  • Python 3.8, PyTorch 1.7.1, torchvision 0.8.2
  • PyQt5 for GUI
  • Conda for environment management
  • GPU: NVIDIA GeForce RTX 3070 Ti

Results and GUI Features

The model achieved an accuracy of 92.04% using SGD with a learning rate of 0.001, 32 epochs, and batch size of 32. The GUI allows users to select datasets, train the model, and predict flower types using a user-friendly interface.

  • Path selection for datasets and model saving
  • Training button to start model training
  • Prediction feature for flower identification
