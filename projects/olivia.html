IABiLSTM-NET Deep Learning Model for Predicting Solar Cell Degradation using Thermal Images

Abstract

ECG-ECANet is a specialized deep learning model developed to address the challenges of accurately detecting heart disease across various ECG datasets. It combines customized data processing techniques for both ECG signals and images, enabling it to handle the unique complexities of ECG data, particularly the multi-label classification issue. This issue arises when a single ECG sample exhibits features of multiple heart conditions. ECG-ECANet is designed not only to improve detection accuracy but also to provide interpretability, making its classification decisions transparent and understandable to clinicians, which supports their trust in using AI for heart disease diagnosis.
