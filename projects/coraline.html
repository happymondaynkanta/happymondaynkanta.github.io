Flower Images Recognition Based on Deep Convolutional Neural Network

Abstract

Flowers are ubiquitous and have both ornamental and functional value. This project develops a computerized flower image recognition system using a ResNet-18 deep convolutional neural network to classify ten flower species. Trained on ~1000 Kaggle images using PyTorch, it achieves 96% training accuracy and 84% validation accuracy. A Tkinter-based GUI was also developed for public use.

Key Contributions

  • Collected a balanced dataset of 10 flower classes from Kaggle (1000 images).
  • Built and trained a CNN model (ResNet-18) using PyTorch with data augmentation.
  • Designed a GUI using Tkinter for easy flower image classification.
  • Achieved 96% training and 84% validation accuracy with improved generalization.

Conclusion

The project successfully demonstrates that ResNet-18 is effective for flower recognition, offering strong generalization and a user-friendly GUI, supporting applications in education, botany, and medicine.

Limitation

Limited dataset size and occasional misclassifications due to visual similarity among flower species. Also, real-time classification may be limited by hardware performance.

Future Work

  • Expand dataset to include more flower classes and higher-quality images.
  • Improve GUI functionality and deployment options (e.g., web or mobile).
  • Explore transfer learning with more advanced networks like EfficientNet.
  • Integrate augmented reality or voice-assisted recognition features.
