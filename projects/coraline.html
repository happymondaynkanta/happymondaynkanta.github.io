Flower Image Recognition based on Deep Convolutional Neural Network

Overview

This project by Xinyi Wang (Coraline) focuses on building a flower image recognition system using a deep learning model based on the ResNet-18 architecture. It utilizes a dataset from Kaggle consisting of 1,000 flower images across 10 categories. The images are classified using a convolutional neural network implemented with PyTorch, and the final application includes a GUI built with Tkinter for user interaction.

Key Highlights

  • Used ResNet-18 architecture trained with cross-entropy loss and optimized using Adam optimizer.
  • Achieved 96% training accuracy and 84‚Äì86.8% validation accuracy.
  • Built a GUI with Tkinter that allows image upload and classification display.
  • Conducted evaluation using metrics like precision, recall, F1-score, and confusion matrix.

Dataset

The dataset includes 1,000 flower images equally split into training (60%), validation (20%), and test (20%) sets. Each image was resized to 256x256x3 for model compatibility.

GUI Implementation

A three-step GUI was implemented to recognize and classify flowers. The user selects an image, views it, and receives a classification result with the flower name displayed.

Limitations and Future Work

Some misclassifications occurred due to image quality and similarities among flower types. Future improvements could include larger and more diverse datasets, advanced CNN architectures, and edge deployment for mobile users.
