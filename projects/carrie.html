Key Contributions

  • Designed a novel dual-path hybrid BiLSTM architecture for accurate solar energy forecasting.
  • Utilized satellite-derived cloud motion vectors to address temporal-spatial variability in radiation patterns.
  • Achieved outstanding predictive accuracy with R¬≤ = 0.982 and RMSE = 0.0301 on test data.
  • Integrated Explainable AI elements for transparency and real-world utility.

Conclusion

The hybrid deep learning framework demonstrated exceptional performance in solar energy prediction by learning complex cloud movement and solar irradiance dependencies. Its ability to process both spatial and temporal data efficiently makes it a reliable model for real-world grid applications and smart solar farm management.

Limitations

  • High Test MAPE value (2198%) due to extreme variance in small irradiance values in early morning/night hours.
  • Model accuracy could degrade if satellite imagery lacks consistent quality or timestamps.

Future Work

  • Refine the model using adaptive loss functions to address MAPE instability.
  • Extend the architecture to include meteorological and environmental sensor fusion.
  • Deploy the model as a cloud-based API service for real-time solar farm monitoring.
